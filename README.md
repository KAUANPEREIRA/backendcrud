# Backend - API de Usuários

Este é o backend do sistema de cadastro de usuários, criado com **Node.js**, **Express**, **SQLite** e **Knex.js**.

## 🔧 Tecnologias Utilizadas

- [Node.js](https://nodejs.org/)
- [Express](https://expressjs.com/)
- [SQLite](https://www.sqlite.org/index.html)
- [Knex.js](https://knexjs.org/)

## 🚀 Funcionalidades

- Criar usuários
- Listar usuários
- Atualizar usuários
- Deletar usuários
- CORS habilitado para uso com frontend separado
